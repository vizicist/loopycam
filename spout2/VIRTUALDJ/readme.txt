VIRTUALDJ

“VDJSpoutReceiver.dll” and “VDJSpoutSender.dll” are 32bit plugins developed using the VirtualDJ 8 plugin SDK.

VDJSpoutSender.dll must be copied to the VirtualDJ 8 - Plugins>VideoEffect folder VDJSpoutReceiver.dll must be copied to the VirtualDJ 8 - Plugins>Visualisations folder

The plugins are only compatible with VirtualDJ 8 will not work on VirtualDJ 7 or earlier.

SpoutPanel.exe is used by VDJSpoutReceiver that shows the list of running Spout senders to allow the user to select one and will be detected after Spout installation.
