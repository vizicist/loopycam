FREEFRAMEGL

Spout FreeframeGL plugins provide Spout output and input for host applications that support FreeframeGL

“SpoutReceiver32.dll” and “SpoutSender32.dll” are for 32bit hosts.
“SpoutReceiver64.dll” and “SpoutSender64.dll” are for 32bit hosts.

Spout Sender   - Vers 3.021
Spout Receiver - Vers 3.031

The dlls must be copied into the plugin folder that has been defined for the the host application.

Refer to the Spout pdf manual for more details. For other applications refer to their documentation.
